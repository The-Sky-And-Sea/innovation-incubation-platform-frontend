# 公共业务模块 - 规格说明

## 1. 模块概述

公共业务模块包含跨三端（企业、载体、政务）的通用功能，包括通知中心、申诉管理、政策诉求和账号注销等。

**核心目标**：
- 统一的消息通知系统
- 申诉处理流程
- 政策诉求提交和处理
- 账号注销流程

## 2. 功能详情

### 2.1 通知中心

**功能**：用户接收系统通知和业务提醒

**页面**：
- `/enterprise/notifications`（企业端）
- `/carrier/notifications`（载体端）
- `/gov/notifications`（政务端）

**通知类型**：
| 类型 | 说明 |
|------|------|
| 入驻审核 | 入驻申请状态更新通知 |
| 变更审核 | 重大事项变更状态通知 |
| 政策申报 | 政策申报审核状态通知 |
| 绩效考核 | 考核启动、评分通知 |
| 系统消息 | 系统公告和提醒 |

**通知状态**：
| 状态 | 标识 | 说明 |
|------|------|------|
| 未读 | unread | 用户未查看 |
| 已读 | read | 用户已查看 |

### 2.2 申诉管理

**功能**：用户对审核结果有异议时提交申诉

**页面**：
- `/enterprise/appeals`（企业端）
- `/carrier/appeals`（载体端）
- `/gov/appeals`（政务端）

**申诉类型**：
| 类型 | 说明 |
|------|------|
| 入驻申诉 | 对入驻审核结果申诉 |
| 变更申诉 | 对变更审核结果申诉 |
| 政策申报申诉 | 对政策申报审核结果申诉 |

**申诉状态**：
| 状态 | 标识 | 说明 |
|------|------|------|
| 待处理 | pending | 提交申诉，等待处理 |
| 处理中 | processing | 正在处理 |
| 已通过 | approved | 申诉通过 |
| 已驳回 | rejected | 申诉驳回 |

### 2.3 政策诉求

**功能**：企业提交对政策的建议和诉求

**诉求类型**：
| 类型 | 说明 |
|------|------|
| 政策建议 | 对现有政策的改进建议 |
| 政策需求 | 希望出台的新政策类型 |
| 其他诉求 | 其他相关诉求 |

**诉求状态**：
| 状态 | 标识 | 说明 |
|------|------|------|
| 待处理 | pending | 提交诉求，等待处理 |
| 处理中 | processing | 正在处理 |
| 已采纳 | adopted | 诉求被采纳 |
| 已回复 | replied | 已有回复 |
| 已驳回 | rejected | 诉求被驳回 |

### 2.4 账号注销

**功能**：用户申请注销账号

**页面**：
- `/enterprise/account-deletion`（企业端）
- `/carrier/account-deletion`（载体端）
- `/gov/account-deletion`（政务端）

**注销条件**：
- 当前无进行中的审核业务
- 当前无未完成的申报
- 已确认注销后果

**注销状态**：
| 状态 | 标识 | 说明 |
|------|------|------|
| 待审核 | pending | 提交注销申请，等待审核 |
| 已注销 | deleted | 账号已注销 |

### 2.5 文件管理

**功能**：企业上传和管理业务材料文件

**页面**：`/enterprise/files`

**文件类型**：
- 营业执照
- 商业计划书
- 入孵协议
- 申报材料
- 其他材料

### 2.6 企业信息

**功能**：企业维护基本资料

**页面**：`/enterprise/info`

**信息字段**：
- 企业名称
- 统一社会信用代码
- 所属行业
- 企业规模
- 企业地址
- 法定代表人
- 联系电话

## 6. 业务规则

### 6.1 通知发送
- 审核状态变更时自动发送通知
- 系统公告可手动发送

### 6.2 申诉限制
- 只能在审核结果出具后7天内申诉
- 每个业务只能申诉一次

### 6.3 注销条件
- 需检查是否有进行中的业务
- 需用户确认注销后果
