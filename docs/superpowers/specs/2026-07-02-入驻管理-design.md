# 入驻管理模块 - 规格说明

## 1. 模块概述

入驻管理模块是平台的核心业务模块，实现企业申请入驻、载体审核、政务审批的完整业务流程。

**核心目标**：
- 企业端：浏览载体、提交入驻申请、查看入驻状态
- 载体端：审核企业入驻申请、查看入驻企业列表
- 政务端：审批入驻记录、办理孵化毕业备案

## 2. 业务流程

### 2.1 入驻申请流程

```
企业提交入驻申请
    ↓
载体审核（初审）
    ↓
政务审核（终审）
    ↓
入驻成功 / 被拒绝
```

### 2.2 入驻状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 草稿 | draft | 企业保存但未提交 |
| 待审核 | pending | 已提交，等待载体审核 |
| 载体审核中 | carrier_review | 载体正在审核 |
| 载体已通过 | carrier_approved | 载体初审通过 |
| 载体已拒绝 | carrier_rejected | 载体初审拒绝 |
| 政务审核中 | gov_review | 政务正在审核 |
| 已通过 | approved | 入驻成功 |
| 已拒绝 | rejected | 入驻被拒 |
| 已退回 | returned | 被退回修改 |
| 已毕业 | graduated | 孵化毕业 |
| 已退出 | exited | 退出孵化 |

### 2.3 入驻孵化状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 未入孵 | not_incubated | 未开始孵化 |
| 在孵中 | in_incubation | 正在孵化 |
| 已毕业 | graduated | 已毕业 |
| 已退出 | exited | 已退出 |

## 3. 功能详情

### 3.1 企业端 - 载体浏览

**功能**：企业浏览可申请的孵化载体列表

**页面**：`/enterprise/carriers`

**内容**：
- 载体名称、类型、所在区域
- 载体简介
- 申请入驻按钮

### 3.2 企业端 - 入驻申请

**功能**：企业填写入驻申请材料

**页面**：`/enterprise/incubation/apply`

**申请材料**：
- 企业基本信息（名称、信用代码、行业、规模、地址）
- 商业计划书
- 入驻需求说明
- 相关附件材料

**流程**：
1. 填写入驻申请表
2. 上传相关材料
3. 提交申请
4. 等待载体审核

### 3.3 企业端 - 入驻管理

**功能**：企业查看自己的入驻申请记录和状态

**页面**：`/enterprise/incubation`

**内容**：
- 入驻申请列表
- 申请状态展示
- 详情查看
- 被退回时重新编辑提交

### 3.4 载体端 - 入驻审核

**功能**：载体审核企业入驻申请

**页面**：`/carrier/incubation`

**操作**：
- 查看企业入驻申请详情
- 审核通过
- 审核拒绝
- 退回修改

### 3.5 载体端 - 企业管理

**功能**：载体查看已入驻的企业列表

**页面**：`/carrier/enterprises`

**内容**：
- 已入驻企业列表
- 企业信息查看
- 入驻状态管理

### 3.6 政务端 - 入驻审批

**功能**：政务终审企业入驻申请

**页面**：`/gov/applications`

**操作**：
- 查看待审批申请
- 终审通过
- 终审拒绝

### 3.7 政务端 - 孵化毕业备案

**功能**：政务办理企业孵化毕业备案

**页面**：`/gov/incubation`

**操作**：
- 办理毕业备案
- 办理退出备案

| 页面 | 路径 | 角色 | 说明 |

### 后端接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取载体列表 | GET | `/enterprise/carriers` | 获取可申请的载体 |
| 提交入驻申请 | POST | `/enterprise/incubations` | 提交入驻申请 |
| 获取入驻记录 | GET | `/enterprise/incubations` | 获取企业入驻记录 |
| 获取入驻详情 | GET | `/enterprise/incubations/:id` | 获取入驻详情 |
| 更新入驻申请 | PUT | `/enterprise/incubations/:id` | 更新入驻申请 |
| 载体审核 | POST | `/carrier/incubations/:id/review` | 载体审核 |
| 政务审核 | POST | `/gov/incubations/:id/review` | 政务审核 |

| 办理毕业 | POST | `/gov/incubations/:id/graduate` | 办理毕业 |
| 办理退出 | POST | `/gov/incubations/:id/exit` | 办理退出 |

## 5. 数据模型

### 入驻记录表 (incubation_records)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| enterprise_id | uint | 企业ID |
| carrier_id | uint | 载体ID |
| status | string | 入驻状态 |
| incubate_status | string | 孵化状态 |
| applied_at | timestamp | 申请时间 |
| carrier_reviewed_at | timestamp | 载体审核时间 |
| gov_reviewed_at | timestamp | 政务审核时间 |
| graduated_at | timestamp | 毕业时间 |
| exited_at | timestamp | 退出时间 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

### 入驻申请表 (incubation_applications)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| incubation_id | uint | 关联入驻记录ID |
| enterprise_id | uint | 企业ID |
| business_plan | text | 商业计划书 |
| requirements | text | 入驻需求 |
| status | string | 申请状态 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

## 7. 审核流程

### 7.1 载体审核
- 查看企业申请材料
- 填写审核意见
- 选择：通过 / 拒绝 / 退回

### 7.2 政务审核
- 查看企业申请材料和载体审核意见
- 填写终审意见
- 选择：通过 / 拒绝
