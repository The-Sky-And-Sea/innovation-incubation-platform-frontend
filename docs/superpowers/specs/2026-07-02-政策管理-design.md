# 政策管理模块 - 规格说明

## 1. 模块概述

政策管理模块用于发布、申报和管理创新创业扶持政策，实现政策从发布到企业申报、载体/政务审核的完整流程。

**核心目标**：
- 政务端：发布政策、管理政策、审核申报
- 载体端：查看政策、审核企业申报
- 企业端：浏览政策、提交申报

## 2. 业务流程

### 2.1 政策发布流程

```
政务发布政策
    ↓
政策生效，企业可见
```

### 2.2 政策申报流程

```
企业浏览政策
    ↓
企业提交申报
    ↓
载体初审
    ↓
政务终审
    ↓
申报通过 / 被拒绝
```

### 2.3 政策状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 草稿 | draft | 政务保存但未发布 |
| 已发布 | published | 政策已发布，企业可见 |
| 已下架 | unpublished | 政策已下架 |
| 已过期 | expired | 政策已过期 |

### 2.4 申报状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 草稿 | draft | 企业保存但未提交 |
| 待审核 | pending | 已提交，等待载体审核 |
| 载体审核中 | carrier_review | 载体正在审核 |
| 政务审核中 | gov_review | 政务正在审核 |
| 已通过 | approved | 申报通过 |
| 已拒绝 | rejected | 申报被拒 |
| 已退回 | returned | 被退回修改 |

## 3. 功能详情

### 3.1 政务端 - 政策管理

**功能**：政务发布和管理政策

**页面**：`/gov/policies`

**功能点**：
- 查看政策列表
- 创建新政策
- 编辑政策内容
- 发布/下架政策
- 查看申报记录

**政策内容**：
- 政策标题
- 政策类型（税收优惠、资金支持、场地支持、人才支持等）
- 申报条件
- 支持标准
- 申报材料
- 受理部门
- 申报截止日期

### 3.2 政务端 - 申报审批

**功能**：政务终审企业政策申报

**页面**：`/gov/applications`

**功能点**：
- 查看待审批申报
- 查看申报详情
- 终审操作：通过 / 拒绝

### 3.3 载体端 - 政策列表

**功能**：载体查看政策列表

**页面**：`/carrier/policies`

**功能点**：
- 查看已发布政策列表
- 查看政策详情

### 3.4 载体端 - 申报审核

**功能**：载体初审企业政策申报

**页面**：`/carrier/applications`

**功能点**：
- 查看待审核申报
- 查看企业申报材料
- 初审操作：通过 / 拒绝 / 退回

### 3.5 企业端 - 政策列表

**功能**：企业浏览可申报的政策

**页面**：`/enterprise/policies`

**功能点**：
- 查看已发布政策列表
- 查看政策详情
- 立即申报按钮
- 查看我的申报记录

### 3.6 企业端 - 政策申报

**功能**：企业提交政策申报

**功能点**：
- 填写申报信息
- 上传申报材料
- 提交申报
- 查看申报状态

## 7. 业务规则

### 7.1 政策可见性
- 草稿状态：仅政务可见
- 已发布：所有角色可见
- 已下架/已过期：企业不可见

### 7.2 申报截止
- 超过截止日期后，企业不可申报
- 已提交的申报不受影响

### 7.3 审核流程
- 载体初审后可选择：通过 / 拒绝 / 退回
- 政务终审后可选择：通过 / 拒绝
